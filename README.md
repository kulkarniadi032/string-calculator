# string-calculator

## Example Inputs and Expected Outputs

### Example 1:
- **Input**: `""`
- **Output**: `0`

### Example 2:
- **Input**: `"1"`
- **Output**: `1`

### Example 3:
- **Input**: `"1,5"`
- **Output**: `6`

### Example 4:
- **Input**: `"1\n2,3"`
- **Output**: `6`

### Example 5:
- **Input**: `"//;\n1;2"`
- **Output**: `3`

### Example 6:
- **Input**: `"1,-2,3,-4"`
- **Output**: `Exception: negative numbers not allowed: -2, -4`
